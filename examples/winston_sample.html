Introducion to Weave

This a sample Julia noweb document that can be executed using Weave. Output from code chunks and Winston plots will be included in the weaved document. You also need to install Pweave from Github in order to use Weave.

This documented can be turned into Pandoc markdown with captured result from Julia prompt.

using Weave
weave(Pkg.dir("Weave","examples","winston_sample.mdw"), plotlib="Winston")

You can also include a plot with caption and hide the code:

Random walk.

x = linspace(0, 3pi, 100)
c = cos(x)
s = sin(x)

p = FramedPlot(
         title="title!",
         xlabel="\\Sigma x^2_i",
         ylabel="\\Theta_i")

add(p, FillBetween(x, c, x, s))
add(p, Curve(x, c, color="red"))
add(p, Curve(x, s, color="blue"))
display(p)
