Introducion to Weave

This a sample Julia noweb document that can be executed using Weave. Output from code chunks and PyPlot plots will be included in the weaved document. You also need to install Pweave from Github in order to use Weave.

This documented can be turned into Pandoc markdown with captured result from Julia prompt.

using Weave
weave("examples/julia_sample.mdw")

Capturing figures

plot(t, sinc(t))
xlabel("x")
